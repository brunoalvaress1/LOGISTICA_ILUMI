# 🚚 Logística Ilumi

Sistema web de **logística e controle de frota** desenvolvido para a empresa **Ilumi**, com foco em organização operacional, gestão de motoristas, controle de serviços e monitoramento de veículos.

## 📌 Objetivo

Centralizar e otimizar processos internos da logística, permitindo maior controle sobre operações diárias, check-ins, jornadas, portaria e histórico de movimentações.

---

## ⚙️ Funcionalidades

### 👨‍💼 Gestão de Usuários e Perfis

* Login seguro via Supabase Auth
* Controle de acesso por perfil:

  * Logística (Administrador)
  * Motorista Próprio
  * Motorista Terceiro
  * Portaria

### 🚛 Gestão de Motoristas

* Cadastro de motoristas
* Vinculação com usuários autenticados
* Controle de informações operacionais

### 📋 Gestão de Serviços

* Criação e acompanhamento de serviços
* Associação de motoristas
* Status operacional

### 🕒 Controle de Jornada

* Check-in e check-out
* Registro de KM inicial e final
* Upload de fotos
* Histórico completo de jornadas

### 🚪 Portaria

* Visualização de veículos liberados
* Saída assistida
* Registro com auditoria

### 📊 Histórico e Controle

* Histórico de movimentações
* Consultas operacionais
* Organização de dados em tempo real

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Bootstrap
* Supabase

  * Auth
  * PostgreSQL
  * Storage
  * RLS (Row Level Security)

---

## 🔐 Estrutura de Perfis

| Perfil             | Permissões                 |
| ------------------ | -------------------------- |
| Logística          | Controle total do sistema  |
| Motorista Próprio  | Jornadas, check-in/out, KM |
| Motorista Terceiro | Serviços designados        |
| Portaria           | Liberação e acompanhamento |

---

## 🧠 Diferenciais do Projeto

* Sistema real aplicado em empresa
* Controle operacional completo
* Segurança por níveis de acesso
* Banco de dados em nuvem
* Interface web responsiva
